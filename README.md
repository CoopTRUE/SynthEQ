# SYNTH EQ

A synthetic equities crypto exchange

## Initial Setup

Pnpm is a blazingly fast package manager

```bash
# install pnpm
npm install -g pnpm

# install node modules
pnpm i

# build once to make server.ts work
pnpm build
```

## Develop

Dev command starts `server.ts` and `vite dev`

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Start

```bash
pnpm start
```

> This app runs on railway.app
